# SiteStudio

**SiteStudio** is a visual, object-based website builder inspired by Blender.
Instead of writing HTML/CSS/JS first, users *create objects directly in the browser* (rectangles, text, components), attach behavior, and SiteStudio generates clean, real code in real time.
